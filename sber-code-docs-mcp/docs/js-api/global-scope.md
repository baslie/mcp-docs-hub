# Глобальный скоуп в Code
Обновлено 18 июля 2024

Глобальный контекст выполнения доступен явно по имени `$global` (аналогично объекту `window.document` в браузере).
В глобальном контексте выполнения сохраняются все загружаемые функции и переменные, объявленные непосредственно в JS-файлах.
Объект `$global` доступен для модификации только на стадии инициализаии смартапа, когда выполняется загрузка JS-файлов и выполняются блоки `init`.
После завершения инициализации изменения объекта `$global` невозможны.
