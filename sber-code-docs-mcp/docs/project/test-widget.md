# Отладка приложения в Code
Обновлено 15 декабря 2023
[ ![](https://developers.sber.ru/docs/img/badges/Code.png)Code ](https://developers.sber.ru/docs/ru/va/code/overview)
В Code вы можете отлаживать работу смартапа в редакторе сценариев.
Для этого используйте тестовый виджет, который позволяет отладить диалог, без [тестирования в эмуляторе](https://developers.sber.ru/docs/ru/va/chat/testing) или на устройстве.
При запуске смартапов с [`$rawRequest`](../js-api/variables/request.md) в тестовом виджете будет возникать ошибка, так как смартап не может получить данные ассистента.
Чтобы запустить смартап в тестовом виджете:
  1. На панели слева откройте вкладку **Редактор** > **Сценарии**.
  2. Нажмите кнопку 
![запустить](./test-widget.md)
или используйте сочетание клавиш ALT+W.

Code соберет смартап и запустит его в тестовом виджете:
![тестовый виджет](https://developers.sber.ru/docs/assets/images/test-widget-d0345b86229b9625bc544320f976027d.png)
При работе в тестовом виджете обращайте внимание на [логи сервера](./server-logs.md). Они содержат полезную информацию о события и ошибках в смартапе.
