Обновлено 27 марта 2024

Страница **Обзор проекта** представляет собой дашборд, на котором отображается аналитическая информация в формате отчетов для текущего проекта.
На дашборде доступны отчеты:
  * Уникальные пользователи, сообщения и сессии;
  * Перевод на оператора;
  * Нераспознанные запросы.

Для переключения между отчетами нажмите на название отчета, например _Уникальные пользователи, сообщения и сессии_ , и выберите отчет из выпадающего списка.
Обратите внимание, что данные отчетов отображаются по серверному времени по стандарту `UTC`.
## Уникальные пользователи, сообщения и сессии
Отчет приводит данные по количеству пользователей, сообщений, сессий в зависимости от выбранного периода.
Параметры отчета:
  * _Уникальные пользователи_ — количество уникальных пользователей за выбранный период.

Под уникальными пользователями подразумевается клиент, который имел одну или более сессий со смартапом за указанный период.
  * _Всего сообщений_ — количество всех пользовательских сообщений за выбранный период.
  * _Всего сессий_ — количество всех сессий за выбранный период.

[Подробнее о сессиях](../js-api/session-lifetime-control.md)
> По умолчанию выбранный период составляет последние 7 дней.
Настройка отображения аналитики отчета:
  1. Для изменения периода нажмите на иконку календаря и укажите начало и конец временного отрезка.
  2. Выберите каналы, для которых будет предоставлена аналитика:

  * _Все каналы_ ;
  * _Все каналы, кроме удаленных_ ;
  * _Один или несколько конкретных каналов_.

> По умолчанию в фильтре выбраны _Все каналы_ , в том числе и удаленные. Удаленные каналы помечаются в списке как _Удален_.
  * При наведении курсора на точки графика отображается количество _Сессий_ , _Уникальных пользователей_ или _Сообщений_ за день. Шаг графика 1 день.
  * Нажмите на элемент легенды, чтобы скрыть график.

## Нераспознанные запросы
Отчет представляет количественный анализ нераспознанных запросов пользователей. И позволяет оценить количество нераспознанных запросов в сравнении с общим количеством запросов.
Параметры отчета:
  * _Всего запросов_ — количество запросов за выбранный период, без учета сообщений после перевода на оператора.
  * _Нераспознанных запросов_ — количество нераспознанных запросов за выбранный период.
  * _Запросы по датам_ — график. Отображает количество всех запросов и нераспознанных по дням за выбранный период.
  * _Процент нераспознанных запросов_ — график. Отображает процент нераспознанных запросов от общего числа за период.

Настройка отображения аналитики:
  1. Для изменения периода нажмите на иконку календаря и укажите начало и конец временного отрезка.
  2. Выберите каналы, для которых будет предоставлена аналитика:

  * _Все каналы_ ;
  * _Все каналы, кроме удаленных_ ;
  * _Один или несколько конкретных каналов_.

> По умолчанию в фильтре выбраны _Все каналы_ , в том числе и удаленные. Удаленные каналы помечаются в списке как _Удален_.
  * При наведении курсора на точки графика отображается количество _Всех запросов_ , _Нераспознанных запросов_ или их процентное соотношение за день. Шаг графика 1 день.
  * Нажмите на элемент легенды, чтобы скрыть график.

Настройки для проекта являются общими для всех пользователей проекта.
