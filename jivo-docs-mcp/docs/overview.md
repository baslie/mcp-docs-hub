# API для разработчиков

Jivo предлагает разработчикам широкие возможности для кастомизации своих решений, а также обширный набор инструментов, позволяющий легко интегрировать собственные и сторонние решения для автоматизации бизнес-процессов.

**Widget API**  — позволяет управлять виджетом, который отображается пользователю на сайте. С помощью него можно управлять окном чата, а также передавать произвольную информацию со страницы, на которой размещено окно чата, в приложение оператора (например, можно передать дополнительную информацию о клиенте, которая также будет сохранена в журнале диалогов).

**Webhooks API** — позволяет получать уведомления о наступление определенных событий, например начале или завершении чата, на стороне вашего сервера, а также передавать в Jivo дополнительную информацию для отображения в приложении оператора в ответе на HTTP-запросы.

**Chat API** — позволяет организовать обработку обращений клиентов из любых источников: как из мобильных, так и из десктоп приложений или полностью кастомизированного виджета на сайте. Операторы получают обращения в приложение Jivo полностью аналогично другим каналам связи.

**Bot API** — позволяет подключать бот-платформы к любым каналам связи и направлять входящие диалоги бот-провайдеру для первичной обработки.
