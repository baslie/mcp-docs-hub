# Интенты и распознавание фраз

Интент — ключевая единица NLU-сервиса, объединяющая в себе набор фраз, намерение пользователя и другую метаинформацию.
**SmartApp Brain** — технология определения смысла фразы пользователей чат-ботов. Включает готовые к использованию модели машинного обучения. Позволяет создавать классификаторы из необработанных логов и управлять обучающими выборками.
В этом разделе вы найдете информацию как использовать сервис с помощью обращения к REST API.
