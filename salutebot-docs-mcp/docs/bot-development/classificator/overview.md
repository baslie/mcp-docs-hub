# Классификатор STS

Платформа поддерживает STS (Semantic Text Similarity) классификатор на базе предобученной функции определения семантической близости.
Классификатор позволяет группировать ответы пользователей на предварительно размеченных данных. Иначе говоря, мы можем задать фразу или набор фраз, которые послужат эталоном для сравнения с запросом пользователя. Эти запросы будут проверяться на семантическое соответствие заданным фразам. И если порог такого соответствия достаточно высок, то можно считать, что две реплики относятся к одному и тому же классу. Соответственно, чат-бот будет на них реагировать одинаково.
Основной плюс этого метода в том, что функция ранжирования примеров работает аналогично функции ранжирования паттернов. Таким образом паттерны и фразы-прототипы могут использоваться совместно. Это дает наиболее высокие результаты ранжирования паттернов.
