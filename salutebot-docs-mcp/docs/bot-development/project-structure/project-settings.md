# Настройка проекта

Чтобы настроить параметры проекта, откройте окно редактирования проекта на вкладке **Разработка** > **Настройки проекта**.
В окне редактирования проекта доступны:
  * **Имя** — название проекта.
  * **Описание** — необязательное поле.
  * Вкладка **Размещение** содержит параметры внешнего репозитория с кодом проекта.
  * Вкладка **Дополнительно** содержит дополнительные параметры проекта. Например, возможность включения автопрослушивания после ответа и запись логов чат-бота.
  * Вкладка **Экспорт/Импорт** позволяет, скачать zip-архив проекта или загрузить в проект файлы с вашего компьютера.
Перед импортом убедитесь, что структура проекта соответствует [стандартной](https://developers.sber.ru/docs/ru/salutebot/bot-steps/project-structure/overview), а все необходимые файлы и папки упакованы непосредственно в архив.
  * Вкладки **Классификатор** и **Настройки NLU** содержат параметры работы с инструментом распознавания речи SmartApp Brain.

На вкладке **Классификатор** , в поле **API-ключ Brain** , вы можете получить ключ, необходимый для обращения к SmartApp Brain Direct API.
Чтобы применить заданные параметры, нажмите **Сохранить**.
