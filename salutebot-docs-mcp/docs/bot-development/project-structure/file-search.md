# Поиск по файлам проекта

В редакторе сценариев реализован поиск текста по файлу проекта.
Для поиска:
  1. Авторизуйтесь на платформе Code, перейдите в проект. Нажмите на панели управления _Редактор сценариев_. Выберите файл, в котором будет производиться поиск.
  2. Нажмите сочетание клавиш `Сtrl+F` или нажмите на иконку поиска внизу страницы.
  3. В открывшемся поле введите текст для поиска. Нажмите клавишу `Enter` или кнопку _Enter_.
  4. В текущем файле будут выделены совпадения. Вы можете перемещаться по результатам, нажимая клавиши стрелок _Вверх_ и _Вниз_.

  

#### Настройка поисковой выборки
Включите на поле поиска одну или несколько настроек выборки, кликнув на иконки слева:
  * чувствительность к регистру;
  * поиск по слову;
  * использование регулярных выражений.

![поиск по файлу](https://developers.sber.ru/docs/ru/salutebot/bot-development/project-structure/file-search)
  

#### Горячие клавиши
Вы можете выделить текст в редакторе, нажать сочетание клавиш `Сtrl+F` для поиска выделенного фрагмента.
  * При переходе между файлами поиск не сбрасывается.
  * Если вы находитесь в фокусе поля поиска, сбросить результаты можно по нажатию на `esc` (в этом случае поле поиска закроется) или по клику на крестик.
  * Вы можете перемещаться по результатам поиска, нажимая клавиши стрелок _Вверх_ и _Вниз_.
  * Переключайте фокус из поля поиска в редактор по сочетанию клавиш `Сtrl+E` и обратно в поиск по `Сtrl+F`.
