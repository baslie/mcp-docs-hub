# Настраиваем профиль

Вы можете перейти в раздел настроек по кнопке в верхнем левом углу экрана под названием пространства.
В настройках вам доступно несколько вкладок.
## Профиль
Данные о владельце аккаунта. Изменить личные данные в профиле можно только при регистрации по почте. При регистрации по Сбер ID данные можно изменить только через Сбер ID.
Независимо от способа регистрации можно загрузить иконку профиля — она будет видна всем пользователям пространств.
## Пространства
На вкладке отображаются все пространства, в которых участвует владелец аккаунта. При нажатии на название пространства откроются вкладки с дополнительными настройками.
## Настройки сервисов
В этом разделе можно настроить доступы к проектам.
При выборе Graph и Code становятся доступны следующие настройки:
  * **Действия пользователей** — информация о пользователях и их действиях в проекте, например, авторизация, выгрузка списка клиентов или фраз, создание проекта и т. п. На вкладке доступны фильтры для поиска записей по типу события и проекта.
  * **Пользователи** – информация о пользователях пространства: имя, роль и уровни доступа к проекту.
  * **Справочник меток** — метки, которые позволяют размечать сессии. Вы можете создать произвольное число меток.
  * **Задачи** — метки для настройки задач на доработку проекта.

Подробная информация по настройке [личного кабинета Studio](../registration.md).
