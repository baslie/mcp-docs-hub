# Восстанавливаем версию сценария

Graph хранит историю всех сохраненных изменений сценария чат-бота. Сохраненную версию можно восстановить на любом этапе разработки.
Чтобы восстановить версию сценария чат-бота:
  1. На панели слева выберите раздел **Настройки** → **Сценарии**.
  2. Нажмите кнопку **Версии** в правом нижнем углу редактора — откроется история изменений сценария.
  3. Выберите необходимую версию сценария в списке и нажмите **Восстановить**.

Выбранная версия откроется в редакторе сценариев. В истории изменений выбранная версия отмечена как **Текущая**.
