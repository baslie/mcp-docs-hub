# Настраиваем время хранения диалогов

Логи диалогов содержат запросы пользователей к SaluteBot и доступны в разделе **Диалоги** в Graph и **Аналитика** → **Диалоги** в Code.
Логи диалогов хранятся в течение одного, трех, шести или 12 месяцев.
Вы можете изменить время хранения логов в настройках проекта.
Graph
Code
Чтобы настроить время хранения логов диалогов:
  1. Откройте проект Graph.
  2. Перейдите в раздел **Настройки**.
  3. На вкладке **Базовые настройки** в раскрывающемся списке в разделе **Время хранения логов диалогов** выберите нужный период хранения логов.

Чтобы настроить время хранения логов диалогов:
  1. Откройте проект Code.
  2. Перейдите в раздел **Настройки проекта**.
  3. Выберите вкладку **Дополнительно**.
  4. В раскрывающемся списке **Время хранения** выберите нужный период хранения логов.
