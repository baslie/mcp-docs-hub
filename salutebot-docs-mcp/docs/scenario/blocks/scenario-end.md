# Завершаем сценарий

Блок **Завершение сценария** предназначен для завершения работы чат-бота.
При выполнении этого блока чат-бот удаляет все данные текущей сессии пользователя и завершает диалог с пользователем. При этом окно чат-бота в интерфейсе остается открытым.
