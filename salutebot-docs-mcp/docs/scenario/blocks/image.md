# Загружаем изображения в чат

Блок **Изображение** предназначен для отправки изображений в чат с пользователем.
Изображения необходимо загружать с компьютера.
На один экран сценария можно добавить несколько блоков изображений. Изображения будут показаны в диалоге последовательно, в порядке добавления.
В мобильном приложении Салют изображения обрезаются по вертикали, чтобы полностью заполнить контейнер по ширине.
## Требования к изображениям
  * Размер — менее 5 Mб;
  * Формат — jpg, png или bmp;
  * Рекомендуемое соотношение сторон — 3:2;
  * Рекомендуемое разрешение — больше 1024 × 682 пикселей.

Размер загруженного контента не должен превышать 200 Мб.
