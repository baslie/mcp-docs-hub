# Отправляем данные оператору

С помощью блока **Отправить данные оператору** чат-бот, запущенный в канале Jivo, отправляет оператору контактные данные пользователя, ранее полученные в сценарии.
Чат-бот может отправить оператору имя пользователя, номер телефона и почтовый адрес. Данные можно указывать с помощью переменных и их значений.
Необходимо заполнить хотя бы один атрибут: имя, email или телефон.
Если данные отправлены успешно, срабатывает переход **Данные отправлены**.
Отправку данных оператору нельзя проверить с помощью тестового виджета.
