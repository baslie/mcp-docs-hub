# Добавляем рекомендованные реплики

Чтобы добавить рекомендованные готовые интенты в сценарий:
  1. Откройте проект чат-бота.
  2. В разделе **Статистика** выберите вкладку **Рекомендованные сценарии**.
Вкладка содержит данные о количестве проанализированных диалогов и списки рекомендуемых интентов и реплик.
Сообщение **У вас мало диалогов** говорит о том, что чат-бот не провел достаточного количества диалогов для создания рекомендаций.
  3. В разделе **Добавьте готовые интенты** нажмите кнопку **Добавить**.
Откроется редактор сценария чат-бота.
  4. Добавьте рекомендованные готовые интенты в сценарий с помощью [блока Интенты](../scenario/blocks/intent.md).

## Создание собственных интентов на основе рекомендаций
Рекомендованные реплики для создания собственных интентов отображаются в разделе **Добавьте частые вопросы клиентов в интенты**.
Чтобы создать собственные интенеты на основе рекомендованных реплик:
  1. Откройте проект чат-бота.
  2. В разделе **Статистика** выберите вкладку **Рекомендованные сценарии**.
Вкладка содержит данные о количестве проанализированных диалогов и списки рекомендуемых интентов и реплик.
Сообщение **У вас мало диалогов** говорит о том, что чат-бот не провел достаточного количества диалогов для создания рекомендаций.
  3. В разделе **Добавьте частые вопросы клиентов в интенты** скопируйте реплики, которые хотите использовать для создания собственных интентов.
  4. Сохраните реплики в стороннем файле, например, в блокноте.
  5. Нажмите кнопку **Настроить**.
Откроется редактор сценария чат-бота.
  6. Используйте [блок Интенты](../scenario/blocks/intent.md), чтобы создать собственные интенты и добавить в них необходимые рекомендованные реплики.
